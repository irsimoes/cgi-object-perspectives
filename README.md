# cgi-object-perspectives


'Mouse Wheel' - Change the scale
'W' - Wireframe Mode
'F' - Filled Mode
'Z' - Toggle Z-Buffer
'B' - Toggle Culling

## Commands
  * mouse wheel - scale
  * f - fill with color
  * w - wire mesh
  * b - (de)activate back-face culling
  * z - (de)activate z-buffer
  * awsd / arrows - move

  * If in "Projeção Perspetiva", the object can be moved around using the mouse.
